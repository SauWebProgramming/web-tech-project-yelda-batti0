Bilişim Sistemleri Mühendisliği Öğrencisi & Web Geliştirici

Hakkımda

Sakarya Üniversitesi Bilişim Sistemleri Mühendisliği 2. sınıf öğrencisiyim. Şu anda ağırlıklı olarak Python üzerine çalışıyorum. Geçmişten gelen C++, C# gibi dillerde de temel seviyede bilgi sahibiyim. Son dönemlerde ise özellikle frontend alanına yönelerek web geliştirme üzerine kendimi geliştiriyorum. HTML, CSS, JavaScript ve React gibi teknolojilerle projeler yapmaya çalışıyor, bu alanda daha yetkin hale gelmeyi hedefliyorum. Teknolojiye ve yazılım dünyasına ilgi duyuyor, öğrendiklerimi sürekli pratiğe dökmeye özen gösteriyorum.
